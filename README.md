# Global Well-Being Intelligence Lab

Custom data analytics project for the “Data Analytics with AI” formative assignment. The solution delivers an end-to-end Python ETL and visualisation pipeline focused on exploring how GDP growth and life expectancy interact across countries between 1952 and 2007.

> **Dataset**: Gapminder Five-Year data (public CSV) downloaded from the Plotly datasets repo: https://raw.githubusercontent.com/plotly/datasets/master/gapminderDataFiveYear.csv

## ETL Pipeline

1. **Extract**  
   - Source: Gapminder CSV downloaded via `Invoke-WebRequest` into `data/raw/`.  
   - Notebook Section 1 validates schema, ranges, and duplicate counts.

2. **Transform**  
   - Implemented in `src/etl_utils.transform_gapminder`.  
   - Steps: rename columns, country-level forward/back fill, median imputation, feature creation (`gdp_total`, `life_stage`, GDP/life deltas), sorting, and infinite-value handling.

3. **Load**  
   - `load_gapminder` persists the cleaned frame to `data/processed/gapminder_clean.csv`.  
   - `run_pipeline()` enables automation for CLI or future schedulers.

4. **Automation & Reuse**  
   - Functions expose typed inputs/outputs to plug into other notebooks or APIs.  
   - Validation returns a dataclass for structured logging or JSON serialisation.

## Data Quality & Validation

- **Completeness**: No nulls in critical numeric columns after imputation.  
- **Integrity**: Negative checks for population, GDP, and life expectancy enforced.  
- **Scope**: 1,704 records, 142 countries, years 1952-2007.  
- **Reproducibility**: Validation summary stored in the notebook output; the dataset can be regenerated by re-running the notebook or `python -m src.etl_utils` (future CLI hook).

## How to Reproduce

1. **Clone & create environment**
   ```bash
   git clone https://github.com/riaz-unar/Python-ETL-and-Visualisation.git
   cd Python-ETL-and-Visualisation
   python -m venv .venv
   .venv\\Scripts\\activate
   pip install -r requirements.txt
   ```
2. **(Re)download data**  
   - Either rerun `Invoke-WebRequest` command from setup or let the notebook download via `extract_gapminder`.
3. **Execute notebook**  
   - Launch Jupyter or VS Code, open `notebooks/gapminder_etl_visualisation.ipynb`, run all cells.  
   - Outputs: cleaned CSV, validation report, static PNGs, Plotly HTML.
4. **Present**  
   - Use Insight Highlights (Notebook Section 5) + saved figures for the final presentation.
